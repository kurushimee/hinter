# Hinter

**Hinter** is a short 3D adventure story game. The main character, Hinter, has to travel alone through a desert with only his home trailer, which keeps breaking during the process.

# Dates

**MVP:** 31st January.
**Demo:** mid-February.

# Controls
- **[WASD]** Move
- **[LMB]** Interact
- **[ESC]** Quit Game
